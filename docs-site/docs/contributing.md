# Contribuindo

## Setup rapido

- Frontend: `pnpm dev`
- Backend: `cd backend && ./mvnw spring-boot:run`
- AI Service: `cd ai-service && uvicorn api.main:app --host 0.0.0.0 --port 8001 --reload`
- Docs: `pnpm docs:start`

## Padroes

- Prefira alterar um modulo por PR.
- Mantenha consistencia com os documentos em `docs_projeto/`.
- Atualize o plano em `docs_projeto/PLANO-EXECUCAO.md` ao concluir tarefas.
- Atualize as paginas em `docs/` quando novas features forem entregues.
