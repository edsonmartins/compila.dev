# Vagas

O marketplace ajuda voce a encontrar oportunidades alinhadas com sua stack.

## Como buscar melhor

- Use filtros por nivel, tipo e remoto.
- Leia requisitos antes de aplicar.
- Compare vagas similares para entender expectativas.

## Antes de aplicar

- Atualize seu portfolio com projetos relevantes.
- Garanta que seus links estao corretos.
- Escreva um texto curto explicando seu interesse.
