# Portfolio

O portfolio mostra sua experiencia real e ajuda em vagas e networking.

## O que incluir

- **Descricao curta**: problema + solucao + resultado.
- **Stack usada**: tecnologias principais.
- **Links**: demo e repositorio.
- **Imagens**: screenshots ou gif do projeto.

## Estrutura sugerida

1. Contexto do projeto.
2. Principais features.
3. Desafios tecnicos resolvidos.
4. Aprendizados.

## Dica rapida

Use o portfolio para conectar desafios que voce concluiu com projetos que voce construiu.
